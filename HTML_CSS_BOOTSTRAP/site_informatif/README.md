# Exercice HTML/CSS : créer un site informatif (par binôme)

HTML :
- Trouvez des infos sur les technos à aborder : HTML, HTTP, URL, DNS, CSS
- Créez un site de 6 pages, une par techno + une page d'accueil (index.html)

CSS :
- Ajouter un menu déroulant en HTML et CSS
- Mettre un titre avec une ombre
- Stylisez le site
