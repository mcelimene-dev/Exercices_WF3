HTTP (HyperText Transfer Protocol)

Description

HTTP est un protocole de communication client-serveur développé pour la W3C.
HTTPS est la variante sécurisée pour l'usage SSL ou TLS. Les clients les plus connus sont les navigateurs Web.
Inventeur : Tim Berners-Lee
Année : 1990

Organisation

L'ordinateur utilise le navigateur pour envoyer une requête à un serveur web. Cette requête demande un document (page HTML, fichier CSS). Le serveur cherche les informations, puis interprête le résultat pour finalement envoyer une réponse.
Cette réponse contient les entêtes du protocole HTTP et le contenu demandé.
