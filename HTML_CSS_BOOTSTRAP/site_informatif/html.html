HTML (Hyper Text Markup Language)

Le langage HTML est un langage de balisage conçu pour représenter les pages web.

Evolution du langage

Le HTML est apparu en 1991 et a connu plusieurs versions avant de devenir le langage HTML qu'on connait aujourd'hui
Depuis 2007, HTML5, encadré par l'organisme W3C (World Wilde Web Consortium), intégre de nouvelles fonctionnalités telles que l'audio et la vidéo.

Description du langage

Le langage HTML s'écrit avec des balises. Ces balises peuvent contenir des attributs. Ce sont celles-ci qui intégre des éléments tel qu'un titre ou un paragraphe dans une page web.
