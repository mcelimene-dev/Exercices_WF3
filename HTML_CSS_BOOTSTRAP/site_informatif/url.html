URL (Uniform Resource Locator)

Description

Le sigle URL subtitue informellement l'expression adresse Web (chaîne de caractères utilisée pour adresser les ressources d'internet :

  • Document HTML
  • Image
  • Son
  • Boîte aux lettres électroniques
  • etc...

Usage

L'adresse Web a plusieurs usages :

Hyperliens
chaque hyperlien du web est construit avec une URL
Barre d'adresse
chaque navigateur dispose d'une barre d'adresse affichant l'URL de la ressource consultée
Historique de navigation
les navigateurs web conservent les URL consultées
Pages favorites
les navigateurs web peuvent constituer un liste de favoris d'URL

Fonctionnement

L'URL est une chaîne de caractères combiant les informations nécessaires pour indiquer à un logiciel comment accéder à une ressource Internet.
Il existe deux types d'URL : absolue ou relative.

URL absolue
Elle permet d'indiquer comment accéder à une ressource indépendamment du contexte où elle est précisée.
URL relative
Les protocoles utilisant un chemin hiéarchiaque permettent l'utilisation d'URL relatives (contient ni de protocole ni nom de domaine).
